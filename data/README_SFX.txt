# Sweeps App Sound Effects Pack

Generated, royalty-free/simple synthetic WAV files for the Sweeps app web display.

Recommended folder:
    data/web_assets/sfx/

Files:
- reel_tick.wav          Short reel tick/click
- reel_spin_loop.wav     Loopable reel spin/tick bed
- reel_stop_clunk.wav    Reel stop/clunk
- reveal_chime.wav       Main reveal chime
- favourite_sparkle.wav  Market favourite sparkle
- roughie_sting.wav      Long-odds roughie sting
- button_press.wav       Button click
- jackpot_flash.wav      Big reveal/fanfare
- warning_buzz.wav       Error/blocked payout buzz
- cash_ding.wav          Paid/cash register ding

Browser note:
Most browsers require the user to click Enable Sound once before these can autoplay.

These are synthetic placeholders, not copied from any commercial pokie/casino machine.
