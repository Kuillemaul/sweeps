# Sweeps Visual Assets Pack

Pokie-inspired visual assets for the Sweeps app web display.
These are original generated placeholder assets. They are not Aristocrat-branded.

Recommended copy location inside the app:

    data/web_assets/

Folders:
- data/web_assets/theme/
- data/web_assets/textures/
- data/web_assets/sprites/

Key files:
- theme/background_1920x1080.png
- theme/cabinet_overlay_1920x1080.png
- theme/reel_frame_large.png
- theme/reel_glass_highlight.png
- theme/flemington_sweeps_logo.png
- theme/badge_favourite.png
- theme/badge_well_fancied.png
- theme/badge_roughie.png
- theme/badge_bolter.png
- textures/brushed_gold.png
- textures/dark_chrome.png
- textures/red_velvet.png
- sprites/led_glow_gold.png
- sprites/spark_gold.png

Use the overlay PNG on top of the /cup3d page, and the background PNG as the body background.
